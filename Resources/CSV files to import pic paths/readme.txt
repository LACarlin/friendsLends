itempaths.csv should be imported into the items table 
- browse to the csv file in the import screen 
- set the format dropdown to CSV
- check the box 'Update date when duplicate keys found on import (add ON DUPLICATE KEY UPDATE)
- enter the following in the column names field:
	iid,itempic
- Click Go

userpaths.csv should be imported into the users table
- browse to the csv file in the import screen 
- set the format dropdown to CSV
- check the box 'Update date when duplicate keys found on import (add ON DUPLICATE KEY UPDATE)
- enter the following in the column names field:
	uid,userpic
- Click Go
